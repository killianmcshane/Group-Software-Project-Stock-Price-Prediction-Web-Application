Welcome to G4SP!

Group 4 Stock Predictions aims to predict short term stock prices by using sentiment analysis and machine learning.

We aim to use our Web-scraping algorithm along with fine tuned machine learning, to accuratly predict the short term price fluctuations of five stocks. With hopefully more coming in the future!

Features

Sentiment analysis

Web-scraping algorithm created using Twitter’s Developer API allowing us to collect sentiment scores of Tweets relating to 5 pre-selected stocks.

Machine Learning

Machine learning model, trained on historical Yahoo Finance data and sentiment scores to output stock price predictions.

Database

Using Google’s ‘Sheets API’ to upload daily stock price predictions to, and allow the web application to pull data from.

Web Application

Web application displaying each of our pre-selected stocks’ current price, along with our prediction for its closing price and a simple graph.

Machine learning

Predicting stock price predictions depends on various factors such as the global economy, company financial reports and performance as well as the current political landscape. Typically, there are two approaches that can be used for predicting stock prices for organisations, technical analysis and fundamental analysis. We will be focusing on technical analysis which is a method that uses historical prices of stocks to predict future prices. A machine learning algorithm uses large quantities of data to find patterns. It learns with experience and can be fine-tuned to suit the given data.

LSTM Algorithm
